## Instruções

- Clone o projeto
  `https://github.com/matheuscostadesign/nlw5-podcastrnext.git`

- Instale as dependências:
  `npm install`

- Execute o servidor:
  `npm run server`

- Iniciar ambiente de desenvolvimento:
  `npm run dev`

- Gerar para produção
  `npm run build`

- Executar como produção
  `npm start`
